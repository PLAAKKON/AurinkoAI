# AurinkoAI

AurinkoAI is an experimental AI-assisted space weather visualization and forecasting project.

## Data source
The example dataset included in this repository (AurinkoAI_data.xlsx) is based on
publicly available interplanetary magnetic field (IMF) data (Bz component).

Primary sources:
- NOAA Space Weather Prediction Center (SWPC)
- NASA / OMNIWeb
